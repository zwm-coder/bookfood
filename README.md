## 在线点单预约管理系统

> 该项目是在华迪实训期间，与小组成员最终完成的一个Web项目；主要分为两部分，一部分是商家的管理系统，另一部分是系统管理员的管理系统。

  

## 技术栈
- JavaBean
- Servlet
- JSP
- JDBC
- MySQL
- HTML5、CSS3  

## 主要实现功能
#### 商家系统
- 查看店铺
- 查看申请
- 添加店铺
- 编辑菜品
- 删除菜品
- 添加菜品
- 管理个人信息
- 登陆、登出

#### 管理员系统
- 查看数据总览
- 商家账户管理
- 用户账户管理
- 申请管理
- 评价反馈处理
- 登录、登出
